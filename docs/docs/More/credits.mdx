# Credits & Notes

---

## Data Source

All poetry content was sourced from [AllamaIqbal.com](https://www.allamaiqbal.com) — a long-running archive dedicated to preserving Iqbal's work. The English translations used in this API are from their collection.

---

## Public Domain

Allama Iqbal — the ideological founding father of Pakistan, national poet, philosopher, and loving father — passed away on **April 21, 1938**. His work has been in the public domain for decades. IqbalAPI exists to make this heritage more accessible to developers, students, and anyone who wants to build on top of it.

---

## Why This Exists

No structured API for Iqbal's poetry existed. The data was there — trapped in an old HTML website — but inaccessible to developers. This project scraped, cleaned, structured, and served it so that his words can reach further.
